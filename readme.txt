Чтобы запустить проект пропишите в терминале, находясь в папке "reactProject_2-main" npm start, при возникновении ошибки отсутствия "react-scripts", прописать npm install, затем снова прописать npm start.

Данный проект выполнен с ипользованием JS библиотеки React. В нем я использовал react-router-dom, react-hook forms, hook useState. Данный проект поддерживает адаптивность благодаря медиа запросам. Все основные части веб приложения вынесены в отдельные компоненты, в некотрых компонентах данные берутся из отдельных файлов, содержащих массивы с данными, через функцию map.
